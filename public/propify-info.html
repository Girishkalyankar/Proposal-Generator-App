Features

  • Google OAuth + Email/Password login
  • Pincode-gated account activation (monetization)
  • Drag-and-drop proposal editor with auto-save
  • AI content generation (multiple tones)
  • Dynamic pricing tables (qty × price, discount, tax, auto-totals)
  • Template library (save/load/clone)
  • PDF export & download
  • Shareable public proposal links (short 8-char tokens)
  • Email proposals to clients
  • Proposal status tracking (Draft → Sent → Viewed → Accepted/Declined)
  • Dashboard with stats, win rate, value charts
  • Dark/light mode
  • PWA (installable on phone/desktop)
  • Admin panel for access code management
  • Responsive design

Security (21 vulnerabilities fixed)

  • Rate limiting on all sensitive endpoints
  • XSS sanitization (DOMPurify)
  • Input validation (Zod schemas)
  • Ownership checks on all resources
  • Secure token generation (crypto.randomBytes)
  • CSRF protection (NextAuth built-in)
  • Email enumeration prevention
  • SQL injection prevention (Prisma parameterized queries)

Deployment

  • Platform: Netlify
  • Build: Auto-deploy from GitHub main branch
  • SSL Certificate: Auto-provisioned by Netlify (Let's Encrypt)
  • Domain: propify-ai.netlify.app (Netlify subdomain)
  • CDN: Netlify Edge (static assets cached)
  • Serverless Functions: All API routes run as Netlify Functions
  • Environment Variables: AUTH_SECRET, GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET, DATABASE_URL, GROQ_API_KEY

Database (Neon)

  • Provider: Neon PostgreSQL (free tier)
  • Region: us-east-1
  • Database: neondb
  • Tables: User, Account, Session, VerificationToken, Proposal, ProposalSection, Template, TemplateSection, AccessCode

Performance Notes

  • Neon free tier has cold starts (2-5s after 5min idle)
  • Retry logic added on public view page + dashboard layout
  • Loading skeletons on all pages
  • CDN caching for static assets

Monetization Model

  • Access codes generated by admin
  • Users must enter a valid code to activate account
  • Codes are single-use, revocable
  • Admin controls all distribution
